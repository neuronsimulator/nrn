41 /*
42 Because python types have so many methods, attempt to do all set and get
43 using a PyHocObject which has different amounts filled in as the information
44 is gathered with a view toward ultimately making a call to hoc_object_component.
45 That requires that array indices or function arguments are on the stack.
46 The major variant is that we may be at the top level. The function arg
47 case is easy since they all come as a tuple in the call method. The
48 array indices come sequentially with a series of calls to the
49 sequence method. A nice side effect of intermediate objects is the extra
50 efficiency of reuse that avoids symbol lookup. Sadly, the scalar case does
51 not give this since the value is set/get instead of returning the
52 intermediate.
53 */

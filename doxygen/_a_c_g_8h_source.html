26 //
27 // Additive number generator. This method is presented in Volume II
28 // of The Art of Computer Programming by Knuth. I've coded the algorithm
29 // and have added the extensions by Andres Nowatzyk of CMU to randomize
30 // the result of algorithm M a bit by using an LCG & a spatial
31 // permutation table.
32 //
33 // The version presented uses the same constants for the LCG that Andres
34 // uses (chosen by trial & error). The spatial permutation table is
35 // the same size (it's based on word size). This is for 32-bit words.
36 //
37 // The ``auxillary table'' used by the LCG table varies in size, and
38 // is chosen to be the the smallest power of two which is larger than
39 // twice the size of the state table.
40 //
41

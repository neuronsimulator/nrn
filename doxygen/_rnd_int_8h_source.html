34 // RandomInteger uses a random number generator to generate an integer
35 // in a specified range. By default the range is 0..1. Since in my
36 // experience random numbers are often needed for a wide variety of
37 // ranges in the same program, this generator accepts a new low or high value
38 // as an argument to the asLong and operator() methods to temporarily
39 // override stored values
